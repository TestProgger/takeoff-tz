# BGS TechWork

## Устанавливаем необходимые зависимости
    npm install

## Для тестового запуска проекта:

### 1. Запускаем сервер для разработки
    npm run dev



## Для запуска продакшен проекта:

### 1. Запускаем сборку проекта:
    npm run build

### 2. Запускаем продакшен проект
    npm run start
