# Тестовое задание 

```bash
# Установка небходимых зависимостей
$ npm install

# Запуск сервера в режиме разработки на хосте localhost:3000
$ npm run dev

# На этом шаге можно остановиться и перейти в браузере на адресс  localhost:3000


# Сборка проекта для продакшена и запуск сервера
$ npm run build
$ npm run start

# Генерация статического проекта
$ npm run generate
```
